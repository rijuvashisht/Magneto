# Magneto AI Skill

## Commands

- `/magneto-analyze` — Analyze codebase and build knowledge graph
- `/magneto-query "text"` — Query the knowledge graph
- `/magneto-plan <task>` — Generate execution plan
- `/magneto-telepathy` — Auto-discover and execute tasks

## When to Use

- Starting work on a new task
- Implementing features that touch multiple files
- Fixing bugs requiring cross-file analysis
- Need to find relevant code quickly
