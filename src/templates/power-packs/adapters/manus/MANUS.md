# Magneto AI — Manus Integration

This project uses **Magneto AI** integrated with Manus API.

## Quick Start

```bash
# Analyze codebase
magneto analyze

# Query knowledge graph
magneto query "auth flow"

# Sync to Manus
magneto adapter sync manus
```

## Configuration

API credentials stored in `.magneto/adapters/manus/config.json` (gitignored).
