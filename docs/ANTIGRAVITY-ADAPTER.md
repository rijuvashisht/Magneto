# Google Antigravity Adapter

Integrate Magneto AI with Google Antigravity IDE.

## Installation

```bash
magneto adapter install antigravity
```

Creates:
```
.agents/
├── magneto/
│   └── ANTIGRAVITY.md
├── skills/
│   └── magneto/
│       └── SKILL.md
└── magneto-adapter.json
```

## Usage

- `/magneto-analyze` — Analyze codebase
- `/magneto-query "text"` — Query knowledge graph
- `/magneto-plan <task>` — Generate plan
- `/magneto-telepathy` — Auto-execute tasks
